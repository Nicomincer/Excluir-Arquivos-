# Excluir-Arquivos-

**Informações**
Programa para excluir arquivos com dada extensão, nome e etc. 

**Como usar**

Para excluir um arquivo, primeiro, coloque o diretório dele em Nome do diretorio. Depois, coloque no *Espera* a extensão, nome, qualquer coisa que identifique o arquivo que você quer excluir . Depois clique em apagar. 

**OBS** 

Ele irá excluir todos os arquivos que tiverem o mesmo valor em *Espera*.
